# 轧制计划管理系统 - 打包说明

## 一、环境准备

### 1. 安装Python
确保已安装Python 3.8或更高版本

### 2. 安装依赖库
```bash
pip install -r requirements.txt
```

依赖库包括：
- pandas
- xlrd
- xlwt
- xlutils
- pyautogui
- openpyxl

### 3. 安装PyInstaller
```bash
pip install pyinstaller
```

## 二、打包方法

### 方法1：使用spec文件打包（推荐）
```bash
python build_with_spec.py
```

### 方法2：直接使用PyInstaller命令
```bash
pyinstaller furnace_order_manager.spec --clean
```

### 方法3：使用简单打包脚本
```bash
python build_exe.py
```

## 三、打包后的文件结构

```
dist/
└── 轧制计划管理系统.exe

使用时需要以下文件在同一目录：
├── 轧制计划管理系统.exe
├── 计划号/
│   ├── 装炉顺序.xls
│   ├── 总计划号列表.xls
│   ├── backup/
│   └── *.xls
├── APS.txt
├── export_coordinates.json
├── processed_plans.txt (自动生成)
├── printed_plans.txt (自动生成)
└── no_aps_plans.txt (自动生成)
```

## 四、使用说明

1. 将 `dist/轧制计划管理系统.exe` 复制到任意目录
2. 确保以下文件/文件夹在同一目录：
   - 计划号文件夹（包含装炉顺序.xls等）
   - APS.txt
   - export_coordinates.json
3. 双击运行exe文件
4. 首次运行可能需要管理员权限

## 五、注意事项

1. **文件大小**：打包后的exe文件通常在30-50MB之间，这是正常的
2. **首次运行**：首次运行时，Windows可能会提示安全警告，选择"仍要运行"即可
3. **防火墙/杀毒软件**：某些杀毒软件可能会误报，请添加信任
4. **更新程序**：如需更新程序，只需替换exe文件即可，无需重新配置
5. **数据文件**：计划号文件夹和配置文件不会被包含在exe中，需要单独管理

## 六、常见问题

### Q1: 打包后运行报错找不到文件
A: 确保计划号文件夹、APS.txt和export_coordinates.json在exe文件同一目录

### Q2: exe文件过大
A: 这是正常的，因为包含了所有Python依赖库。可以使用UPX压缩减小体积

### Q3: 运行时闪退
A: 可能是缺少某些文件或权限问题。尝试以管理员身份运行

### Q4: 杀毒软件报毒
A: 这是误报，可以添加到信任列表或暂时关闭杀毒软件

## 七、高级选项

### 减小exe文件大小
在spec文件中添加UPX压缩：
```python
exe = EXE(
    ...
    upx=True,  # 启用UPX压缩
    ...
)
```

### 添加图标
准备一个.ico格式的图标文件，修改spec文件：
```python
exe = EXE(
    ...
    icon='app.ico',  # 添加图标
    ...
)
```

### 显示控制台（调试用）
修改spec文件：
```python
exe = EXE(
    ...
    console=True,  # 显示控制台窗口
    ...
)
```

## 八、技术支持

如遇到打包问题，请检查：
1. Python版本是否正确
2. 所有依赖库是否已安装
3. 文件路径是否正确
4. 是否有足够的磁盘空间
