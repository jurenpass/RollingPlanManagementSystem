# 轧制计划管理系统 - 使用说明

## 一、程序位置

打包后的exe文件位于：`dist\轧制计划管理系统.exe`

文件大小：109.78 MB

## 二、部署说明

### 1. 文件准备
将以下文件/文件夹复制到目标目录：

```
目标目录/
├── 轧制计划管理系统.exe      (主程序)
├── 计划号/                     (文件夹)
│   ├── 装炉顺序.xls
│   ├── 总计划号列表.xls
│   ├── backup/                  (备份文件夹)
│   └── *.xls                   (计划号文件)
├── APS.txt                     (APS钢种列表)
└── export_coordinates.json       (坐标配置文件)
```

### 2. 首次运行
1. 双击 `轧制计划管理系统.exe`
2. 如果Windows提示安全警告，点击"更多信息" → "仍要运行"
3. 首次运行可能需要管理员权限

### 3. 自动生成的文件
程序运行后会自动生成以下文件：
- `processed_plans.txt` - 已处理的计划号列表
- `printed_plans.txt` - 已打印的计划号列表
- `no_aps_plans.txt` - 包含无APS钢种的计划号列表

## 三、功能说明

### 主要功能
1. **自动导出** - 自动导出装炉顺序、总计划号列表和计划号明细
2. **处理计划** - 处理选中的计划号，提取指定字段
3. **显示计划** - 打开选中的计划号文件
4. **打印计划** - 打印选中的计划号
5. **刷新数据** - 刷新计划号列表

### 状态标注
- `[已处理]` - 计划号已处理
- `[已打印]` - 计划号已打印
- `[无APS]` - 计划号包含不在APS列表中的钢种
- `[无文件]` - 计划号对应的文件不存在

## 四、注意事项

1. **文件路径**：确保所有必需的文件和文件夹在exe文件同一目录
2. **文件占用**：程序运行时不要手动修改计划号文件夹中的文件
3. **备份**：建议定期备份计划号文件夹和配置文件
4. **更新**：如需更新程序，只需替换exe文件即可

## 五、常见问题

### Q1: 运行时提示找不到文件
A: 检查计划号文件夹、APS.txt和export_coordinates.json是否在exe文件同一目录

### Q2: 程序闪退
A: 可能是权限问题，尝试以管理员身份运行

### Q3: 杀毒软件报毒
A: 这是误报，可以添加到信任列表

### Q4: 无法打开Excel文件
A: 确保系统已安装Excel或WPS

## 六、技术支持

如遇到问题，请检查：
1. 文件路径是否正确
2. 是否有足够的磁盘空间
3. 是否有足够的权限
4. 系统是否满足最低要求

最低系统要求：
- Windows 7 或更高版本
- 4GB RAM
- 500MB 可用磁盘空间
